# Login interno DonCor

Este documento registra o plano de implementação do login interno do sistema.

## Etapas

1. Exibir campo de credencial na tela de entrada.
2. Exibir campos de credencial inicial no cadastro de colaborador.
3. Permitir atualização de credencial em Configurações.
4. Conectar a validação ao backend antes do uso em produção.

## Observação

Valores reais de credenciais não devem ser versionados no repositório. Use variáveis de ambiente ou persistência segura no backend.
